  • Basic Paragraph:
    This is a normal paragraph.
  • Heading 1:
    #This paragraph is Heading 1
  • Heading 2:
    ##This paragraph is Heading 2
  • Heading 3:
    ###This paragraph is Heading 3
  • Bold Paragraph:
    **This paragraph is bold**
  • Italic Paragraph:
    *This paragraph is italic*
  • Bold + Italic:
    ***Bold and italic text***
  • Lists (Ordered & Unordered):
    - Item 1
    * Item 2
    1. First item
    2. Second item
  • Link:
    This is a [link](https://example.com)
  • Images:
    [Alt text](image-url "Optional title")
  • Code Blocks:
    Use backticks:
    `inline code`

    Use triple backticks for multi-line:
    ```
    code block
    ```
  • Blockquote:
    > This is a quoted paragraph
  • Highlighting Text:
    ==highlighted text==
  • Subscript:
    H~2~O
  • Superscript:
    E=mc^2^
  • Mathematical Expressions:
    Inline: $a^2 + b^2 = c^2$
    Block: $$\int_0^\infty e^{-x} dx = 1$$
  • Strikethrough:
    ~~This text is crossed out~~
  • Horizontal Line:
    --- or ___
  • Tables:
    | Name | Age |
    |------|-----|
    | Alice | 24 |
    | Bob | 27 |
  • Comments (Invisible):
    [//]: # (This is a comment)
